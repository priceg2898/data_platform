| Command | Description |
| -------- | -----------|
| `poetry run python -m etl.env_vars_test.main ` | Run the python.exe from the current folders venv |
| `poetry run python -c "import pydantic_settings; print('OK')" ` | Verify package installation |
| `poetry add/remove package_name` | add/remove a package |
| `poetry env remove --all  ` | Remove venv |
| `poetry init --name shared --no-interaction` | Initialise poetry in a directory |
